Role Information

Information

Note: When you create a new user, a temporary password will be generated and sent to their email address.

Available Roles:

  • Customer - Can create and view their own tickets
    • Customer - Regular customer (requires manager email)
    • Manager - Can approve/reject tickets from customers
  • Agent - Can work on tickets
    • Normal - Standard agent (consultant)
    • Delivery Lead - Can assign tickets to agents
    • HOD - Head of Department
    • Admin - System administrator

Important: To create a normal customer, you must first create a Manager in the system, then use that manager's email when creating the customer.
